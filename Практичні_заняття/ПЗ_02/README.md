# ПЗ 02 — Розгортання OpenVPN

## Мета

Налаштувати сервер та клієнт OpenVPN з використанням сертифікатів.

## Завдання

1. Встановити OpenVPN та Easy-RSA
2. Створити PKI та сертифікати (CA, сервер, клієнт)
3. Налаштувати OpenVPN-сервер
4. Підключити клієнта та перевірити з'єднання
5. Налаштувати маршрутизацію трафіку через VPN

## Обладнання та програмне забезпечення

- 2 віртуальні машини Linux (Ubuntu 22.04 або аналог)
- OpenVPN 2.5+
- Easy-RSA 3
