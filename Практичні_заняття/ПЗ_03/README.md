# ПЗ 03 — Конфігурація WireGuard

## Мета

Налаштувати VPN-з'єднання за допомогою протоколу WireGuard.

## Завдання

1. Встановити WireGuard на Linux
2. Згенерувати пари ключів для сервера та клієнта
3. Налаштувати інтерфейси wg0 на обох вузлах
4. Перевірити з'єднання та маршрутизацію
5. Налаштувати автозапуск WireGuard

## Обладнання та програмне забезпечення

- 2 віртуальні машини Linux (Ubuntu 22.04 або аналог)
- WireGuard (kernel module + tools)
