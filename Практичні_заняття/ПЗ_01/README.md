# ПЗ 01 — Налаштування IPsec VPN

## Мета

Набути практичних навичок налаштування IPsec VPN між двома вузлами.

## Завдання

1. Встановити та налаштувати StrongSwan/Libreswan на Linux
2. Створити конфігурацію Site-to-Site IPsec тунелю
3. Налаштувати IKEv2 з pre-shared key
4. Перевірити встановлення з'єднання та передачу трафіку
5. Проаналізувати IPsec-трафік за допомогою Wireshark

## Обладнання та програмне забезпечення

- 2 віртуальні машини Linux (Ubuntu 22.04 або аналог)
- StrongSwan або Libreswan
- Wireshark
