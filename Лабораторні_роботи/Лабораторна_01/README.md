# Лабораторна робота 01 — Дослідження VPN-трафіку

## Мета

Дослідити та проаналізувати мережевий трафік VPN-протоколів за допомогою аналізатора пакетів.

## Завдання

1. Захопити трафік при встановленні IPsec-з'єднання
2. Проаналізувати пакети IKE-обміну
3. Захопити зашифрований ESP-трафік
4. Захопити та проаналізувати трафік OpenVPN
5. Порівняти структуру пакетів різних VPN-протоколів
6. Скласти порівняльну таблицю характеристик

## Обладнання та програмне забезпечення

- Wireshark
- VPN-стенд з лабораторних занять
- Linux/Windows з мережевим інтерфейсом у promiscuous mode

## Критерії оцінювання

| Критерій | Бали |
|----------|------|
| Захоплення та аналіз IPsec трафіку | 30 |
| Захоплення та аналіз OpenVPN трафіку | 30 |
| Порівняльна таблиця | 20 |
| Висновки | 20 |
