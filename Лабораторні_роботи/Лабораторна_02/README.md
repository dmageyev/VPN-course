# Лабораторна робота 02 — Порівняння VPN-протоколів

## Мета

Провести порівняльний аналіз продуктивності та безпеки різних VPN-протоколів.

## Завдання

1. Налаштувати тестовий стенд з трьома VPN-рішеннями: IPsec, OpenVPN, WireGuard
2. Виміряти пропускну здатність кожного протоколу (iperf3)
3. Виміряти затримку (ping, traceroute)
4. Оцінити навантаження на CPU при шифруванні
5. Проаналізувати параметри безпеки кожного протоколу
6. Зробити обґрунтовані висновки щодо вибору протоколу

## Обладнання та програмне забезпечення

- 3 віртуальні машини Linux (Ubuntu 22.04)
- IPsec (StrongSwan), OpenVPN, WireGuard
- iperf3, ping, htop

## Критерії оцінювання

| Критерій | Бали |
|----------|------|
| Налаштування стенду | 20 |
| Вимірювання пропускної здатності | 25 |
| Вимірювання затримки | 15 |
| Аналіз навантаження CPU | 20 |
| Висновки та рекомендації | 20 |
