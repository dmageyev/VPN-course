# Лекція 07 — Безпека та аутентифікація у VPN

## Мета

Вивчити методи забезпечення безпеки та аутентифікації у VPN-системах.

## План

1. Моделі аутентифікації (Pre-shared keys, сертифікати, MFA)
2. Управління ключами
3. Типові вразливості VPN
4. Захист від атак (Man-in-the-Middle, Replay-атаки)
5. Аудит та моніторинг VPN
