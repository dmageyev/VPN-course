# Лекція 07 — Сучасний протокол WireGuard та концепція Zero Trust (ZTNA)

## Мета

Вивчити протокол WireGuard и концепция Zero Trust (ZTNA) у VPN-системах.

## План лекції

1. WireGuard: философия минимализма и высокая производительность.
2. Криптографический стек WireGuard: ChaCha20, Poly1305 и Noise Protocol.
3. Концепция Cryptokey Routing и отсутствие состояний (stateless).
4. Кризис традиционных VPN: переход к Zero Trust Network Access (ZTNA).
5. Архитектура ZTNA: микросегментация, брокеры доступа и непрерывная проверка.
