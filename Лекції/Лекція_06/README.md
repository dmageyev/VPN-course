# Лекція 06 — SSL/TLS VPN та архітектура OpenVPN

## Мета

Вивчити SSL/TLS VPN та їх застосування для захищеного доступу.

## План лекції

- История и эволюция SSL/TLS 
- VPN: преимущества перед IPsec.
- Протокол TLS: процесс Handshake, наборы шифров и защита данных.
- Архитектура OpenVPN: работа в User-space и использование TUN/TAP.
- Сравнение производительности: TCP против UDP режимов.
- Методы многофакторной аутентификации (MFA) в SSL VPN.
