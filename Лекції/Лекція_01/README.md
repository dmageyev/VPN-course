# Лекція 01 — Основи кібербезпеки та введення в технології VPN

## Мета

Розглянути основи кібербезпеки та сучасні технології VPN. Вивчити тріаду CIA, принципи криптографії та класифікацію VPN-рішень.

## План лекції

1. Тріада CIA (Confidentiality, Integrity, Availability) як основа безпеки.
2. Криптографічне хешування та контроль цілісності даних.
3. Симетрична криптологія: алгоритми AES, ChaCha20 та режими шифрування.
4. Визначення та класифікація VPN (Site-to-Site, Remote Access).
5. Рівні моделі OSI в контексті тунелювання.
