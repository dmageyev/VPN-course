# Лекція 03 — Мережева архітектура та механізми тунелювання

## Мета

Вивчити мережеву архітектуру Overlay/Underlay, механізми інкапсуляції, віртуальні інтерфейси TUN/TAP та особливості маршрутизації у VPN-тунелях.

## План лекції

1. Концепція Underlay та Overlay мереж. 
2. Механізми інкапсуляції та накладні витрати (Overhead).
3. Віртуальні інтерфейси: драйвери TUN та TAP.
4. Проблеми MTU, MSS та фрагментація пакетів у тунелях.
5. Маршрутизація: Full-tunnel проти Split-tunnel.
