# Лекція 08 — Огляд хмарних VPN-рішень. Моніторинг, діагностика та аналіз інцидентів безпеки

## Мета

Ознайомити студентів з сучасними хмарними VPN-рішеннями та тенденціями розвитку.

## План лекції

1. Хмарні VPN: AWS VPN, Azure VPN Gateway, Google Cloud VPN
2. SD-WAN та його зв'язок з VPN
3. Zero Trust Network Access (ZTNA)
4. VPN у контексті DevOps та мікросервісів
5. Перспективи розвитку VPN-технологій
6. Метрики качества VPN-соединений: Latency, Jitter и Packet Loss.
7. Инструменты синтетического мониторинга и реального анализа (iPerf3, MTR).
8. Типовые неисправности: проблемы с MTU, DNS-утечки и ошибки аутентификации.
9. Кейс-стади: анализ массовых взломов VPN Ivanti и Pulse Secure (2024).
10. Лучшие практики эксплуатации: Hardening шлюзов и управление патчами.
