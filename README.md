# Навчальний курс: Віртуальні приватні мережі (VPN)

Цей курс призначений для студентів бакалаврату спеціальності **«Інфокомунікації та мережна безпека»**. Програма поєднує фундаментальні принципи криптографічного захисту інформації з практичними навичками розгортання та адміністрування сучасних VPN-рішень.

## Структура навчального курсу (Кредитів: 6.0)

Програма розрахована на 8 лекцій (по 2 год.), 3 практичні заняття (по 2 год.) та 2 лабораторні роботи (по 4 год.).

| Розділ | Кількість | Годин |
|--------|-----------|-------|
| Лекції | 8 | 16 |
| Практичні заняття (ПЗ) | 3 | 6 |
| Лабораторні роботи | 2 | 8 |
