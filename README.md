# VPN-course

Курс лекцій присвячений вивченню технологій віртуальних приватних мереж (VPN). У курсі розглядаються принципи роботи, протоколи, налаштування та практичне застосування VPN-рішень.

## Структура курсу

| Розділ | Кількість |
|--------|-----------|
| Лекції | 8 |
| Практичні заняття (ПЗ) | 3 |
| Лабораторні роботи | 2 |
